# :material-alert-decagram-outline: Should I use Windows 10 or Windows 11 with AtlasOS?

Windows 11 is now required for the latest release of AtlasOS. 

AtlasOS development prioritizes compatibility and optimization for the **latest supported Windows versions**.

*   **Primary Focus:** Windows 11 is now our primary focus for development, testing, and new features.
*   **End of Atlas Support for Windows 10:** AtlasOS has **cease support for Windows 10** in alignment with Microsoft's end-of-life date (October 14, 2025).

## :material-check-decagram-outline: Use Windows 11

Using Windows 11 as the base for your AtlasOS installation is crucial for:

*   Receiving ongoing security updates from Microsoft.
*   Ensuring the best compatibility and performance with current and future AtlasOS releases.
*   Maintaining a secure and stable system.
